# synthonyx-kit-macros

Procedural macros for the Synthonyx Kit.

**Phase 1: empty stub.** The crate exists so the published name is reserved
and the workspace topology is complete. Phase 3 will populate it with:

- `compose_runtime!` — the kit's macro DSL for composing RTMs into a
  runtime. See [the architecture overview](../../docs/architecture.md) and
  the project plan for the Phase 3 roadmap.

In Phase 1, runtime composition is done by hand —
see [`docs/writing-an-rtm.md`](../../docs/writing-an-rtm.md) and the
`manual_runtime` example in the facade crate.
